# TechZone Business Process Support & Power BI Analytics Project

A business process support and retail analytics project based on the **TechZone Electronics (PVT) Ltd** case study. This project uses Excel-based retail data and Power BI reporting to support business process improvement, KPI tracking, operational analysis, and data-driven decision-making.

The project was developed as part of an academic Business Process Support module and focuses on how a growing electronics retailer can move from manual spreadsheet-based operations toward a more modern, data-supported business process.

## Project Overview

TechZone Electronics is a consumer electronics retailer headquartered in Colombo, Sri Lanka, with multiple branches across the country. The company sells products such as mobile phones, laptops, smart TVs, home appliances, and accessories.

The business currently faces several operational challenges due to manual handling of customer transactions, complaints, inventory movement, supplier details, and branch-level reporting through Excel spreadsheets.

These challenges include:

* Delays in reporting and decision-making
* Human errors in manual data handling
* Miscommunication between branches
* Difficulty tracking customer complaints and returns
* Limited visibility into branch performance
* Difficulty identifying stock and inventory issues
* Lack of real-time business insights

This project demonstrates how business data can be used to support better operational reporting, performance monitoring, and process improvement through a Power BI dashboard.

## Project Objectives

The main objectives of this project are to:

* Analyse TechZone’s sales, inventory, complaints, and return data
* Identify key business performance indicators
* Build a Power BI dashboard for decision support
* Present branch-level and product-level insights
* Highlight customer issue patterns using complaint and return data
* Support business process improvement through data visualisation
* Demonstrate how BI tools can improve operational visibility

## Dashboard Pages

The Power BI report includes the following pages:

| Page                      | Purpose                                                                                                   |
| ------------------------- | --------------------------------------------------------------------------------------------------------- |
| Dashboard Overview        | Provides a summary view of overall business performance                                                   |
| KPI Cards                 | Displays key metrics such as total sales, total quantity, average price, complaint ratio, and return rate |
| Total Sales Forecast      | Shows sales trends and forecasted performance over time                                                   |
| Branch Performance        | Compares sales and performance across TechZone branches                                                   |
| Product Category Insights | Analyses performance by product category                                                                  |
| Customer Issues           | Highlights complaints and return-related concerns                                                         |
| Inventory Analysis        | Supports stock and inventory-related decision-making                                                      |
